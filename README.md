# portfolio-reccos

A portfolio recommendation application built with React and Vite.

## Getting Started

### Development
```bash
npm run dev
```

### Testing
```bash
npm test          # Run tests in watch mode
npm run test:run  # Run tests once
npm run test:ui   # Run tests with UI
```

### Build
```bash
npm run build
npm run preview
```
